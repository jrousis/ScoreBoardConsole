# ScoreBoardConsole

ESP32-based LVGL scoreboard console project.

## Overview
- UI built with LVGL.
- Display driver via Arduino GFX.
- Wi-Fi configuration stored in EEPROM.

## Build
Use the Arduino IDE or compatible toolchain targeting ESP32 (C++14).

## Usage
- Configure Wi-Fi credentials in the UI.
- Power the device and follow on-screen controls.

## Notes
Hardware-specific pin mappings are defined in `ScoreBoardConsole.ino`.
